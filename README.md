# FinSight 💰

AI-powered expense tracker with chatbot insights.

## 🚀 Features
- Receipt scanning
- Expense tracking
- AI chatbot (Gemini)
- Dashboard insights

## 🛠️ Setup

### Backend
cd backend  
npm install  
Create .env file:
GEMINI_API_KEY=your_key  

Run:
node server.js  

---

### Frontend
cd frontend  
npm install  
npm run dev  

---

## 🌐 Run
Frontend → http://localhost:5173  
Backend → http://localhost:5000
